Simple by design

Three steps to a full project

No accounts, no setup, no cost. Just an idea and a click.

1
Describe your idea
Write a sentence or two about what you want to build. Be as specific or vague as you like.
2
AI generates the blueprint
CursedPlan-AI sends your idea to Llama 3.3 on Groq and parses the response into six structured modules.
3
Explore, refine, export
Browse all six tabs, refine with feedback, re-evolve, then export a GitHub-ready ZIP or Plan MD file.

Getting Started
Clone the repo, add your Groq API key to src/js/config.js, open with Live Server, and you're running.
→ Read the guide
 🧠
AI Pipeline
The flow runs through promptBuilder.js → aiClient.js (Groq) → responseParser.js (3-strategy JSON extraction).
→ Read the guide
 🎨
UI System
Six render modules in src/js/ui/ — each tab has its own renderer. CSS uses glassmorphism with CSS custom properties.
→ Read the guide
 🔑
API Key Setup
Copy src/js/config.example.js to src/js/config.js and add your free Groq API key. The file is gitignored automatically.
→ Read the guide
 📦
Export System
githubExporter.js builds a full GitHub repo ZIP including README, LICENSE, CI workflow, issue templates, and all code files.

Evolution & Code Gen
evolutionEngine.js and codeGenerator.js make separate LLM calls to expand your project with improvements and deeper code.
