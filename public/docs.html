Getting Started

What is CursedPlan-AI?

CursedPlan-AI is a free, open-source AI tool that turns any plain-text idea into a complete software project blueprint — in seconds, straight from your browser.

Paste in your idea and CursedPlan-AI generates an architecture diagram, a step-by-step development roadmap, a folder structure, starter code files, and AI-powered improvement suggestions — all in one place. No account needed, no backend, no cost.

It runs entirely client-side. The only thing leaving your machine is the API call to Groq (which you authorize with your own free API key). There is no CursedPlan-AI server.

💡 Free to use: Groq's free tier for Llama 3.3-70B is more than enough for dozens of project generations per day at zero cost.

Getting Started

Using the App

CursedPlan-AI is live on GitHub Pages — no installation, no account, no API key needed. Just visit and go.

  1. Visit the app at ai-alon.github.io/CursedPlan-AI and click ⚡ Launch the Engine.
  2. Type your idea — describe any software project you have in mind. The more specific you are, the better the result.
  3. Click ⚡ Generate Project and wait a few seconds while the AI builds your full blueprint.
ℹ️ CursedPlan-AI is free and open source. We cover the AI costs using our own free Groq subscription so you don't have to set anything up. If you run into an error — the community loves this tool and the free tier sometimes gets busy. Wait a moment and try again. ❤️

Features

Everything CursedPlan-AI can do out of the box:

FeatureHow to use it
Generate a project Type any idea into the textarea and click ⚡ Generate Project (or press Ctrl+Enter)
✏️ Refine After generating, click Refine and tell the AI what to change — e.g. "Switch backend to Python"
🔮 Evolve Get fresh AI suggestions for improving the project — runs independently, can be re-run any time

Contributing

Contributions are very welcome — bug fixes, new features, better prompts, improved docs, or even a typo fix. Every pull request matters.

💡 First time contributing to open source? CursedPlan-AI is a great place to start. It's pure vanilla JS — no build step, no framework. If you can read the code, you can contribute.

How to open a PR

  1. Fork the repository on GitHub and clone your fork.
  2. Create a branch for your change: 
    git checkout -b feat/your-feature-name
# or
git checkout -b fix/your-bug-fix
  3. Make your changes and verify them before opening the PR.
  4. Commit with a clear message. We follow Conventional Commits: feat:, fix:, docs:, chore:.
  5. Open a Pull Request against the main branch and describe what you changed and why.

What to contribute

TypeExamples
Bug fixesDiagram rendering issues, JSON parsing edge cases, UI glitches
New featuresNew export formats, settings panel, keyboard shortcuts
Prompt improvementsBetter AI prompts for higher quality output
UI / UXAccessibility, mobile layout, animations
DocsClearer explanations, missing info, screenshots

Ground rules

  • Keep PRs focused — one change per PR is much easier to review
  • Test your change end-to-end before opening a PR
  • Don't add npm dependencies — the project intentionally has zero build tooling
  • Never hardcode API keys in any file that gets committed
  • Be respectful in reviews and discussions — this is a welcoming community
ℹ️ Not sure what to work on? Browse the open issues — anything labelled good first issue is a great starting point.

Reporting Issues

Found a bug or something that doesn't work right? Please open a GitHub issue — good reports get fixed fast.

Before you report

  • Check the FAQ below — your issue may already be covered
  • Search existing issues to avoid duplicates
  • Make sure you're using Live Server and not opening the file via file://
  • Open DevTools (F12) → Console and check for any error messages

Suggesting a feature

Have an idea? Open a GitHub issue with the enhancement label. Describe what you want and why it's useful. The fastest way to get a feature added is to open a PR implementing it yourself.

FAQ

Is CursedPlan-AI free?

Completely free. No account, no credit card, no setup. We cover the AI costs ourselves using a free Groq subscription so the community can use it without any friction.

Do I need to create an account or enter an API key?

No. Just open the app and start generating. Everything is handled for you.

I'm getting an error when generating

This usually means our free Groq subscription hit its rate limit — this happens when many people are using the tool at the same time. Wait 30–60 seconds and try again. If the daily limit is reached, it resets at midnight UTC.

❤️ If you're hitting errors often, it means the community is loving this tool. We're doing our best to keep it free and running for everyone.

The AI generated a bad or incomplete project

Try clicking ✏️ Refine and describing what's wrong — e.g. "Give this project a better name" or "Add more detail to the roadmap". You can also regenerate from scratch with a more specific idea description.

The diagram panel is blank

The AI occasionally returns a diagram that can't render. Click ✏️ Refine and ask "Fix the architecture diagram" to regenerate it.

My project history disappeared

History is saved in your browser's local storage. Clearing browser data or switching browsers will wipe it. The app stores up to 10 projects — oldest ones are dropped automatically.

Can I save or share my project?

Yes — use 📦 Export to download a full GitHub-ready ZIP, or 📄 Plan MD to download a Markdown file you can share or drop into any LLM (ChatGPT, Claude, Gemini) to keep building.

Can I fork and host my own version?

Yes. Fork the repo on GitHub and deploy to GitHub Pages, Netlify, or Vercel. See the Contributing section for details.
